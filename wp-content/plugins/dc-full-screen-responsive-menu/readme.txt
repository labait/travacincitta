This WordPress plugin makes it very easy to create a responsive full screen or overlay menu for your your WordPress website. 

== Description ==

Our highly customizable plugin / enables you to create a full screen responsive menu, with several possible effects. This menu would be an overlay over all your content, which by default makes it both very trendy (this is one of the hottest trends in web design right now) and also full reponsive, since this works well on all types of devices, from smartphones to desktops. The customization options of this menu allow you to create colors and styling for any website's theme, since you have full color customization options. Also, with the integration of Google fonts, you are able to use any Google font, simply by specifying the name of the font. To top it all off, you can decide the direction of the the animation of the full screen menu overlay. 

This is a Full Screen Responsive WordPress plugin provided by [CollectiveRay](https://www.collectiveray.com/ "Your favorite WordPress plugins")

The full screen WordPress menu plugin offers the following features:

* Shows a full screen responsive menu in WordPress
* Integrates directly with WordPress menu functionality
* Customizable animation (swing in from left, right, top or bottom of screen)
* Customizable background colors
* Customizable font (using Google font - just specify the name of the Google font)
* Customizable font color

== Installation ==

To install the module, simple go to (Plugins > Add New> Upload Plugin), choose the file you have downloaded above and click on the upload and install button. Use it by going to Apperance > DC Full Screen Menu and specifying all the options as necessary, and then enable this menu in your Theme
